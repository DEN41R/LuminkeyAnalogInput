Контроллер

Включить контроллер

Ваша клавиатура Magger68 может работать как контроллер, передавая аналоговые сигналы клавиш.

Включить клавиши клавиатуры

Включите цифровые клавиши, если вы хотите использовать их для набора текста. Выключите цифровые клавиши, чтобы они не мешали кнопкам вывода контроллера (Gamepad Mode).

Привязка контроллера

Добавьте кнопки контроллера, перетащив их на клавиатуру. Щелкните правой кнопкой мыши по клавише клавиатуры, чтобы удалить кнопку.

Интерактивная аналоговая кривая

Перетягивайте точки на графике для настройки мертвых зон.

Отклик контроллера

Включить регулировку угла

Изменяет вектор диагонального направления.

Использовать Квадратный Вектор джойстика

Обеспечивает абсолютный максимальный диапазон по диагоналям.

Продвинутые настройки клавиш

Настройте Snap Tap, Rapid Trigger и DKS (Динамические нажатия) для конкретных клавиш.

Snap Tap (Взаимоисключающие клавиши)

При нажатии двух противоположных кнопок активной становится последняя нажатая.

Rapid Trigger

Сбрасывает нажатие клавиши при отпускании, не дожидаясь возврата в начальную точку.

DKS (Dynamic Keystrokes)

Привязка разных действий к глубине нажатия (до 4 точек).

Тестер Геймпада

Просмотр текущего аналогового вывода (режим контроллера должен быть включен).

Необходимые драйверы

Для работы программы требуются следующие драйверы. Пожалуйста, установите их и перезапустите программу.

После скачивания и установки обязательно перезагрузите компьютер!
